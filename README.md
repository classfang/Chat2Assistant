# Chat2Assistant

![demo](/demo/demo.png)

An assistant-centered ChatGPT electron client with a minimalist interface will support more models in the future

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
